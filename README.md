# lista_filmes

Aplicativo Flutter para listagem de filmes populares, construido durante um curso de Flutter.

## Funcionalidades

- Lista de filmes populares
- Tela de detalhes do filme
- Interface com Material Design 3

## Tecnologias

- Flutter
- Dart
- Material Design 3

## Getting Started

```bash
flutter pub get
flutter run
```

## Estrutura do Projeto

```
lib/
├── main.dart
├── app.dart
├── home/               # Tela principal com lista de filmes
├── filme_detail/       # Tela de detalhes do filme
└── shared/             # Theme e componentes compartilhados
```
